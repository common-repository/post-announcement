== Description ==
Through this plugin, user can able to show the announcement or notice to users based on each post.

* Author can able to show the announcement or notification about that post under the each post.
* Notification post will be scrolled and stopped on mouse hover.
* Admin can able to select the background and font color of notification in dash board.
* Customize title can be set for Announcement.

For more information : http://buffercode.com/wordpress-post-announcement-plugin/

== Installation ==
1. Upload the "post-announcement" directory to the plugins directory.
2. Go to the plugins setting page and activate "Post Announcement". 
3. Go to Settings --> Post Announcement --> Fill the Custom heading, then select the font and background color
4. Users can able to view "Post Announcement" in Posting area with scrolling effect

For more information : http://buffercode.com/wordpress-post-announcement-plugin/
